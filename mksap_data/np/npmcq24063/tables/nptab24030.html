Pharmacologic Causes of Tubulointerstitial Nephritis

Analgesics

Analgesic nephropathy (e.g., aspirin, caffeine, acetaminophen, NSAIDs [including COX-2 inhibitors], or combinations)

Antibiotics

Penicillins (e.g., amoxicillin, methicillin, oxacillin)

Cephalosporins (e.g., cefazolin, cephalexin, cefoxitin)

Fluoroquinolones (e.g., ciprofloxacin, levofloxacin)

Sulfonamides (e.g., trimethoprim-sulfamethoxazole)

Macrolides (e.g., azithromycin, clarithromycin)

Rifampin

Gastrointestinal Therapeutics

Proton pump inhibitors (e.g., omeprazole, pantoprazole)

H 2 blockers (e.g., ranitidine)

5-Aminosalicylates (e.g., mesalazine, sulfasalazine)

Sodium phosphate (can cause phosphate nephropathy)

Calcineurin Inhibitors

Cyclosporine

Tacrolimus

Antineoplastic Agents

Immune checkpoint inhibitors (e.g., atezolizumab, ipilimumab)

Tyrosine kinase inhibitors (e.g., sorafenib, sunitinib)

Ifosfamide

Pemetrexed

Lenalidomide

Vemurafenib

Cisplatin

Nitrosoureas (e.g., streptozotocin, semustine)

Drugs of Abuse

Synthetic cannabinoids

Anabolic androgenic steroids

Toluene

Cocaine

Other

Antiretrovirals (e.g., abacavir, indinavir, tenofovir)

Allopurinol

Diuretics (e.g., thiazide, loop, potassium sparing)

Lithium

Antiepileptic drugs (e.g., carbamazepine, phenytoin, phenobarbital)

Antihypertensives (e.g., ACE inhibitors, angiotensin receptor blockers, calcium channel blockers)

Prolonged exposure to any medication that can cause acute interstitial nephritis
